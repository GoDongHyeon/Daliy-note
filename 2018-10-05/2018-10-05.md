### React study
*toFixed* : 메서드는 고정 소수점 표기법을 사용하여 숫자를 반환합니다.
Ex)
```cpp
num = 12345.6789;
Num.toFixed(); // return ‘12346’
Num.toFixed(1); //return ‘12345.7’
num.toFixed(6); //return ‘12345.678900’
```
