# cy.clock()



이 함수는 Date() 객체의, 'setTimeout', 'clearTimeout', 'setInterval', 'clearInterval' 등을 제어할 수 있다.

```cpp
cy.clock()
cy.clock(now)
cy.clock(now, functionName)
cy.clock(options)
cy.clock(now, options)
cy.clock(now, functionName, options)
```

now          | Number |
functionName | Array  |
options      | Object |
