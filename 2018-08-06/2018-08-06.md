```
- 객체를 활용해서 만든 구구단 구현
- 배열과 문자열 메소드

    - splice(s_index, numelement) : s_index 부터 numElement 갯수만큼 자라냄
    - split : 문자열을 배열로 나누고 싶을 때 사용하는 메소드
- switch-case 조건문
- 객체 생성자 이용하기
- 참조 변수의 동작 방식
    - 일반 변수는 '값'만 전달/ 참조 변수는 자체가 분신
- prototype : 각 객체가 공통적으로 가지고 있는 속성값을 한번에 입력함

    - add, commit, push, branch, merge, checkout
- 사칙연산 (변수, 조건문, 반복문 사용)
```

### GIT 기본 개념 수업 ###
```
pwd : Print Working Directory(현재 디렉터리 이름)
ls -al : (현재 디렉터리에 있는 모은 파일의 권한, 날짜 등을 볼 수 있음)
cd : Change Directory (디렉터리 이동)
mkdir : Make Directory (디렉터리 생성)
mv : move (파일 이동)
rm : remove (디렉터리 삭제)

Git bash로 파일 등록시 
1. git add .
2. git commit -m '메세지'
3. git push origin master
```
